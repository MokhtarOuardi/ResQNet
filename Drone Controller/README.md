# 🎮 ResQNet — Drone Controller

> **Drone swarm coordination and flight control module for the ResQNet platform.**

The Drone Controller manages real-time communication with the drone swarm, handling flight path coordination, feed collection, and waypoint management.

---

## Responsibilities

- Receive geolocation bounds and mission parameters from the Reasoning Agent
- Coordinate drone swarm formation and flight paths
- Stream live drone video feeds back to the Reasoning Agent
- Handle altitude adjustments for Scout → Search phase transitions
- Manage drone safety protocols and return-to-base procedures
